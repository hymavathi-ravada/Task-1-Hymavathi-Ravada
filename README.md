# Data Cleaning & Preparation

# project Overview
This project focuses on cleaning and preparing raw sales data for analysis using Python, Pandas, Excel, and VS Code. The dataset contained missing values, duplicate records, inconsistent text formatting, and data quality issues that were identified and corrected through a structured data-cleaning pipeline.

# Project Objective
The main objective of this project is to improve data quality and prepare the dataset for accurate analysis by:
* Handling missing values
* Removing duplicate records
* Standardizing text fields
* Correcting data formats
* Validating numerical calculations
* Exporting a cleaned dataset for further analysis
  
# Tools & Technologies
* Python
* Pandas
* NumPy
* Excel
* VS Code

# Steps Performed
1. Data Loading
2. Data Exploration
3. Missing Value Handling
4. Duplicate Removal
5. Column Standardization
6. Text Cleaning
7. Date Formatting
8. Data Validation
9. Data Export

# Key Features
* Identified and handled missing values in the dataset.
* Removed duplicate records to improve data quality.
* Standardized text fields and column names.
* Corrected data formats for dates and numerical values.
* Performed data validation checks to ensure accuracy.
* Exported a clean and analysis-ready dataset for further use.

# Output
The cleaned dataset is successfully generated and saved as:
cleaned_dataset.xlsx

After the data cleaning and validation process, the dataset is free from missing values, duplicate records, and invalid entries. All columns have been standardized and formatted correctly, making the data reliable and ready for analysis.
